# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-26

### Added

- Initial release
- Core `Flex` builder with Row, Column, Container, Scroll
- Basic controls: Text, Button, Input, Toggle, Slider, Image, Dropdown, Progress, Foldout
- Spacing utilities: Spacer, Gap, Divider
- `FlexContainer` with chainable styling methods
- `VisualElementExtensions` for fluent API on any element
- `FlexTheme` ScriptableObject for design tokens
- Static `Theme` accessor for colors, spacing, typography
- Pre-built `FlexComponents`: Cards, Buttons, Lists, Modals, Forms, Navigation, Status
- `FlexScreen` base class for screen management
- `FlexScreenManager` for navigation with stack
- `FlexStylesheet` USS generator
- Basic Usage sample
- Game Menu sample with multiple screens
- Full documentation
